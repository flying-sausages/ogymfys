# ogymfys
ogymfys stands for "ogymfys gets your music from your server"


## Dependencies
* ffmpeg
* rsync
* parallel

Please install these

## Installation
* Clone this
* cd into directory
* check the perms are good for execution
* run once
* edit config in ~/.ogymfys/config

## Updating
* git pull
